


### 使用

1. `git clone https://github.com/Liting1/electron-vue-template.git`
2. `npm install`
3. `npm run dev` 开发
4. `npm run build` 生产打包




### 功能
1. 热加载开发
2. 打包生成App

### 项目目录结构

```
|—— app						webpack 构建输出文件目录
|—— pack					打包软件输出目录
|—— builder					webpack配置文件目录
|—— config					配置文件目录
|—— src 					项目资源目录
|	 |—— main 				主进程目录
|	 |—— pages 				其他页面目录
|	 |—— renderer	  		渲染进程目录
|—— .babelrc 				Babel配置文件
|—— .gitignore
|—— package-lock.json
|—— package.json
|—— README.md

```
